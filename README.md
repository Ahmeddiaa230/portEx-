# PortEx Rate Calculator

An intelligent freight management platform simulating a real export shipment management system. Built with Vanilla JS, CSS, Node.js, Express, and SQLite.

## Setup

1. \`npm install\`
2. \`node server/index.js\`
3. Open your browser and navigate to \`http://localhost:3000/calculator.html\`

## Features

- **Rate Calculator**: Two modes (CBM Input vs Dimensions Input) with real-time preview.
- **Customs Duty Toggle**: Integrated HS Code verification and calculations.
- **Internationalization**: Full support for English, Arabic, and Turkish with RTL layout flipping.
- **Responsive Sidebar**: Collapsible CSS Grid layout for Desktop, sliding overlay for Mobile.
- **Persisted History & Analytics**: SQLite integration to save calculations, load stats, and manage history.
- **200+ Ports**: Searchable dropdowns for exact rate generation.
